## Simulação Lúdica do Projeto (FPGA via Web)

Este trecho do projeto foi reservado para orientar a simulação lúdica do funcionamento do sistema.

Para visualizar o funcionamento via web, siga os passos abaixo:

### ✅ 1º Passo – Acesso à Plataforma

Acesse o link: **https://pitanga.inplace-da.com/pt** e faça login.

> Caso não tenha cadastro, crie uma conta gratuitamente e selecione o plano **Community**.

### ✅ 2º Passo – Abrir o Ambiente de Criação

Após o login, acesse o ambiente de criação do Pitanga, que permite reproduzir o comportamento de uma FPGA.

### ✅ 3º Passo – Inserir os Arquivos

- No campo superior, copie e cole o conteúdo do arquivo **`MODULO_SEMAFORO.v`**
- No campo inferior (pinagem), copie e cole o conteúdo do arquivo **`Pinout.txt`**

### ✅ 4º Passo – Executar

Pronto! Agora é só rodar a simulação e observar o funcionamento.
